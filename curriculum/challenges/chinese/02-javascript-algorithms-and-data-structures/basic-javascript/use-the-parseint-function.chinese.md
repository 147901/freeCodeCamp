## Description
<section id="description"> <code>parseInt()</code>函数解析字符串并返回一个整数。这是一个例子： <code>var a = parseInt(&quot;007&quot;);</code>上面的函数将字符串“007”转换为整数7.如果字符串中的第一个字符无法转换为数字，则返回<code>NaN</code> 。 </section>

## Instructions
